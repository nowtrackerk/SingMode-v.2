SingMode Command

Windows 11 Deployment Guide | v1.0.2

This guide provides step-by-step instructions for deploying SingMode Command as a professional karaoke workstation on Windows 11.

1. System Requirements

Component Minimum Recommended
Processor Intel Core i5 Intel Core i7 (12th Gen+)
RAM 8GB 16GB
Wi-Fi 2.4GHz 5GHz (for lowest latency)

2. Installation & Setup

01 Environment: Download and install Node.js (LTS) from nodejs.org. Ensure it matches your Windows arch (x64).
02 Launch: Open your SingMode folder, enter the Windows directory, and double-click start_dev_server.bat.
03 Installer: To create a standalone .exe, run build_installer.bat. This creates a native installer for production use.

3. Network Configuration

For smartphone microphones to function, your PC and guest phones must be on the same network profile.

  • Set your Wi-Fi profile to "Private" in Windows Settings.
  • Configure your Network IP in the DJ Console UI to match your local IP (e.g., 192.168.1.100).
